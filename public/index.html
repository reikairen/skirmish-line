How to Use Skirmish Line

Overview

Two participants compete to secure markers along a line of 9 nodes. Each participant is dealt 6 tiles from a shared pool of 54 (values 1-9 across 6 categories). The objective is to secure 5 of the 9 nodes, or 3 adjacent nodes in a row.

Each Cycle

  • Select a tile from your tray by clicking it (it will lift up).
  • Assign it to a node by clicking the numbered marker on the workspace. Each side of a node holds up to 3 tiles.
  • After assigning, click "End Cycle" to draw a new tile from the pool and pass to the other participant.

Securing Nodes

When both sides of a node have 3 tiles, it is automatically evaluated and secured by the participant with the stronger formation. Formations are ranked as follows:

RankFormationDescription
5Category SequenceSame category + consecutive values
4Triplet3 tiles with the same value
3Category SetSame category, any values
2Sequence3 consecutive values, any category
1TotalSum of tile values (tiebreaker)

If two formations share the same rank, the one with the higher total value secures the node. Equal formations result in no assignment.

Session End

The session ends when one participant secures 5 nodes total, or 3 nodes in an unbroken row. If the pool empties and no further assignments can resolve a result, remaining complete nodes are resolved automatically.

Solo Mode

Compete against an automated peer at three levels: Standard (exploratory), Advanced (pattern-aware), or Expert (strategic and adaptive).

Collaborative Mode

Quick Match finds an open session or creates one visible to others. Private Session generates a 4-character code you can share directly. Use "Connect" with a code to join a private session.
